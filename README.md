# PLIT

### 시간 적응형 오디오-비주얼 하이브리드 콘텐츠 플랫폼

**시간이 조각나도, 정보는 완전하게**  
당신의 시간에 맞춰 변하는 콘텐츠 플랫폼

![PLIT 웹페이지](./screenshot.png)

</div>

---

## 문제 인식

> 온라인 강의 완강률, 겨우 **15%**

이동 중, 운동 중, 운전 중... 영상 보기엔 애매한 시간들이 있습니다.  
PLIT은 듣기만 해도 이해되고, 필요할 때만 화면을 보면 됩니다.

## 핵심 기능

<tr>
<td width="50%">

### 시간 적응형 콘텐츠
1분에서 1시간까지, 사용자의 가용 시간에 맞춰 콘텐츠 길이와 형식이 자동 조정됩니다.

</td>
<td width="50%">

### 오디오 우선 설계
화면을 보지 않아도 핵심 내용을 이해할 수 있도록 설계되었습니다.

</td>
</tr>
<tr>
<td width="50%">

### 자동 TTS 생성
텍스트만 입력하면 AI가 자연스러운 한국어 나레이션을 자동으로 생성합니다.

</td>
<td width="50%">

### 크리에이터 친화적
복잡한 편집 없이 텍스트 기반으로 콘텐츠를 제작할 수 있습니다.

</td>

## TTS 생성 프로세스

```python
# 슬라이드별 스크립트 정의
slides = [
    {"id": 1, "script": "온라인 강의 완강률, 겨우 15%예요"},
    {"id": 2, "script": "왜 그럴까요?"},
    ...
]
```

**1단계** → 슬라이드별 스크립트 준비  
**2단계** → 자동 음성 파일 생성  
**3단계** → 웹 앱에서 즉시 재생 가능

### 특징

- ✅ 실행 한 번으로 모든 음성 파일 완성
- ✅ 실시간 진행 상황 표시
- ✅ 자연스러운 한국어 발음
- ✅ 조각난 시간을 완전한 학습으로 전환
